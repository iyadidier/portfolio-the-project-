Yatzy Game Summary

This Yatzy game is an implementation of the classic dice game, designed with HTML, CSS, and JavaScript. It includes features such as:

  • Rolling dice and updating scores
  • Scoring categories including Three-of-a-kind, Four-of-a-kind, Full house, Small straight, Large straight, Yatzy, and Chance
  • Responsive design for mobile and desktop viewing
  • Integration of best practices in JavaScript for managing game state

Technical Documentation

The Yatzy game utilizes the following key technologies and techniques:

  • HTML5: for structuring the game interface.
  • CSS3: for styling and ensuring a visually appealing layout.
  • JavaScript: for dynamic game functionality, including event handling and scoring logic.
  • jQuery: leveraged for DOM manipulation and event handling.
  • JSON & fetch API: utilized for exchanging data between client and server.
  • Node.js & NPM: used for setting up the server-side environment and managing packages.
  • Express.js: chosen for creating the server and defining RESTful APIs.
  • MVC Architecture: adopted this design pattern to separate the game's logic, interface, and data.
  • JavaScript Asynchronous: used async/await with fetch API to handle server responses.
  • VS Code: my preferred code editor, enhancing productivity with its vast array of extensions.
  • GitHub: used as the version control system, facilitating collaboration, code reviews, and project tracking.

Design System Documentation

For a consistent design experience, I've documented all design-related decisions using Markdown, ensuring a clear understanding of design elements like colors, typography, and components.

Highlights & Challenges

During the development process, I faced several challenges:

  • Responsive Design: Ensured the game interface is user-friendly across devices, using CSS Flexbox and Media Queries.
  • Asynchronous Operations: Leveraged JavaScript's asynchronous features to handle dice rolls and score calculations without disrupting the user experience.
  • Server-Client Interaction: Used the fetch API and Express.js to create a seamless flow of data between the client and server.
  • Animation: Employed JavaScript & CSS to bring the dice roll to life, enhancing user engagement.
